Job Description

Skills Required

Qualifications and Experience

Hot beans web is looking for junior web developer to join our team. The ideal candidate will have a strong understanding of web development and be able to work in a fast-paced environment. The candidate will be responsible for developing and maintaining web applications, as well as working with the team to develop new features and functionality. The candidate should have experience with HTML, CSS, JavaScript, and other web technologies.

Courses

React Course

Our React course is designed to help you learn the fundamentals of React and build your own web applications. The course covers everything from setting up your development environment to building complex web applications using React and other modern web technologies.

Duration: 6 weeks

Frontend Course

Our frontend course is designed to help you learn the fundamentals of frontend and build your own website. The course covers everything from setting up your development environment to building websites using HTML, CSS and JavaScript.

Duration: 2 weeks

Backend Course

Our backend course is designed to help you learn the fundamentals of backend and build your own web applications. The course covers everything from setting up your development environment to building complex web applications using Node.js and other modern web technologies.

Duration: 4 weeks

Full Stack Course

Our full stack course is designed to help you learn the fundamentals of full stack development and build your own web applications. The course covers everything from setting up your development environment to building complex web applications using React, Node.js and other modern web technologies.

Duration: 18 weeks
